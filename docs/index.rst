nbcollection & nbcollection-ci
==============================

nbcollection-ci
+++++++++++++++

nbcollection-ci provides a set of tools to streamline the Scientific Review process of Jupyter Notebooks. Designed
through collaboration between astropy, STScI DSMO, and SPB. This repository exists to automate as many administrative
tasks as possbile for an engineer or scientist looking to enhance a Github Pull Request review process.

nbcollection-ci shipping with the ability to install Build Machinery, enhancing your Github Repository with a complete
CI/CD solution utilizing CircleCI. There is planned support for Github Actions.

.. _Github: https://github.com/astropy/nbcollection/issues
